To use this, do the following:
- Install Python (I used 3.9)
- If using GPU, install CUDA / cuDNN (see https://www.tensorflow.org/install/gpu)
- Install the following python packages by running pip install <package> or py -m pip install <package>
    - tensorflow (should install dependencies, I used 2.8)
    - matplotlib
    - opencv-python
    - pillow
- Put any image data in the data folder in the format of the examples in there(like mnist_fashion)
- See examples.txt for some examples
